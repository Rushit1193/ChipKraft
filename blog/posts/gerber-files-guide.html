Understanding Gerber Files: What You Need to Send to Your Fab

A complete breakdown of the Gerber file set — copper layers, drill files, board outline, solder mask, and silkscreen — and how to export them correctly from your EDA tool.

By ChipKraft Team·Mar 10, 2026·7 min read

What Are Gerber Files?

Gerber files are the industry-standard file format for describing PCB layers to a manufacturer. Each file describes one layer of your board — copper, solder mask, silkscreen, board outline — as a set of vector graphics. Together, they form a complete manufacturing package.

The modern format is Gerber X2 (RS-274X extended), which includes embedded metadata. Most fabs also accept the older RS-274X format.

The Complete Gerber File Set

For a standard 2-layer board, you need these files:

  • GTL / F.CuTop copper layer — all pads, traces, and pours on the front of the board
  • GBL / B.CuBottom copper layer — same as above but on the back
  • GTS / F.MaskTop solder mask — defines where solder mask is REMOVED (i.e. where pads are exposed)
  • GBS / B.MaskBottom solder mask
  • GTO / F.SilkSTop silkscreen — component outlines, reference designators, polarity markers
  • GBO / B.SilkSBottom silkscreen
  • GKO / Edge.CutsBoard outline / mechanical layer — defines the exact shape and size of your PCB
  • DRL / .drlDrill file (Excellon format) — all hole positions and sizes, both plated and non-plated

💡 4-layer boards also need G2L (Inner Copper Layer 1) and G3L (Inner Copper Layer 2) files.

Understanding the Drill File

The drill file is separate from the Gerber files — it uses the Excellon format. It defines every hole on your board:

  • PTH (Plated Through Hole) — holes with copper on the walls, used for through-hole components and vias
  • NPTH (Non-Plated Through Hole) — unplated holes, used for mounting holes and mechanical features

Some EDA tools export PTH and NPTH as separate files. Check your fab's requirements — most accept both in a single file with separate tool definitions.

How to Export from KiCad

  1. Open PCB Editor → File → Fabrication Outputs → Gerbers
  2. Set output folder, check all required layers are ticked
  3. Enable "Use Protel filename extensions" for maximum fab compatibility
  4. Click "Plot"
  5. Then click "Generate Drill Files" — set format to Excellon, enable separate PTH/NPTH if required
  6. Zip all output files into a single .zip archive and upload to your fab

"Always run a Gerber viewer check before submitting. Free tools like Gerbv or KiCad's built-in viewer catch layer mix-ups that would otherwise cause a costly re-spin."

Common Mistakes

  • Missing board outline — without Edge.Cuts, the fab doesn't know where to cut your board
  • Wrong solder mask polarity — mask files define openings (where mask is removed), not where mask exists
  • Forgetting NPTH drill file — mounting holes without the NPTH file may be drilled as plated holes
  • Silkscreen over pads — ink on pads causes soldering problems; run a DRC check before export
  • Submitting unzipped files — always zip the entire Gerber set into one archive
