Getting Started: Your First PCB Footprint in Chipkraft

Step-by-step guide to creating your first IPC-standard footprint using the Wizard Builder — from reading the datasheet to exporting a KiCad-ready file.

By ChipKraft Team·Mar 22, 2026·10 min read

Before You Start

You'll need three things: a Chipkraft account (free), the component datasheet, and a few minutes. We'll use a common QFN-16 package as our example — a package you'll encounter constantly in microcontroller and power IC design.

📖 What is a footprint? A PCB footprint is the pattern of copper pads, silkscreen markings, and courtyard outlines on your board that matches a physical component. A wrong footprint = components that don't fit or don't solder correctly.

Step 1 — Read the Datasheet

Open your component's datasheet and find the "Package Dimensions" or "Land Pattern Recommendation" section. You need these values for a QFN-16:

  • Pin count — 16 (4 per side)
  • Pitch — distance between pad centres (e.g. 0.5mm)
  • Package body size — e.g. 3mm × 3mm
  • Pad length and width — from the land pattern table
  • Exposed pad (EP) size — the thermal pad on the bottom (e.g. 1.7mm × 1.7mm)

"Always use the manufacturer's recommended land pattern over generic IPC formulas when one is provided — it accounts for package-specific thermal and electrical behaviour."

Step 2 — Open the Wizard Builder

  1. Log in to Chipkraft and click "Footprint Builder"
  2. Click "New Footprint" → select "Wizard"
  3. In the package selector, choose QFN
  4. Enter your values from the datasheet

Step 3 — Fill in the Parameters

The wizard asks for these fields — here's what each means:

  • Pin Count: Total number of pads around the perimeter (not counting the exposed pad)
  • Pitch: Centre-to-centre distance between adjacent pads
  • Body Width / Height: Overall package dimensions
  • Pad Width / Length: Individual pad dimensions from the land pattern
  • Exposed Pad: Enable this and enter EP dimensions if your package has a thermal pad
  • IPC Density Level: Choose B (Nominal) for most designs; A (Most Material) for hand-soldering; C (Least Material) for ultra-dense boards

Step 4 — Review the Preview

Chipkraft shows a live preview as you type. Check:

  • Pin 1 marker is in the correct corner (usually top-left)
  • Pad numbering goes counter-clockwise from pin 1
  • Courtyard (yellow outline) has at least 0.25mm clearance around all pads
  • Silkscreen reference designator (RefDes) is inside the courtyard, not overlapping pads
  • The exposed pad (if present) is centred and correct size

💡 Common mistake: Forgetting the exposed pad thermal vias. The footprint itself just defines the pad — remember to add thermal vias through the EP in your PCB layout, connecting to the ground plane below.

Step 5 — Export

Click "Export" and choose your format:

  • KiCad (.kicad_mod) — drag into your KiCad project's footprint library folder
  • Altium (.PcbLib) — import via the Altium footprint library manager
  • Eagle (.lbr) — import via Eagle's library manager

You're Done!

Your first footprint is ready to use. The whole process takes under 5 minutes once you're comfortable reading datasheets. As you build more footprints, save them to your Chipkraft library so your whole team can reuse them.
